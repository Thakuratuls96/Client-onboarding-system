# Make.com Scenario Overview

### Workflow

- **Trigger:** Google Sheets (Watch New Rows)
- **AI Processing:** Google Gemini AI (Generate Welcome Message)
- **Failure Handling:** Send Admin Email Notification
- **PDF Generation:** PDF.co (HTML to PDF)
- **File Storage:** Google Drive
- **Client Email:** Gmail (Send Onboarding PDF)
- **Status Update:** Google Sheets (Mark Client as Onboarded)

This scenario automates the complete client onboarding process using AI, from generating personalized welcome messages to delivering onboarding documents with built-in error handling.
